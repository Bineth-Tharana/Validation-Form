# ✅ Validation Form

A responsive and user-friendly **Validation Form** built using **HTML, CSS and JavaScript**. This project validates user input in real-time, ensuring that all required fields are completed correctly before form submission. It helps improve user experience and data accuracy by displaying meaningful error messages and validation feedback.

## 🚀 Features

* 📋 User-friendly form interface
* ✅ Real-time input validation
* 📧 Email format validation
* ⚠️ Custom error messages
* 🎨 Responsive design for all devices
* 🚫 Prevents form submission when inputs are invalid

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 How It Works

1. User enters information into the form fields.
2. JavaScript validates each field based on predefined rules.
3. Error messages appear if the input is invalid.
4. Once all fields are valid, the form can be submitted successfully.

## 💡 Validation Checks

* Required fields cannot be empty.
* Email must follow a valid format.
* Invalid inputs display appropriate error messages.
